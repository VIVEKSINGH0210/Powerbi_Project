📊 Amazon Sales Dashboard (Power BI)

An interactive and visually appealing Amazon Sales Dashboard built using Power BI. This project focuses on analyzing sales performance, profit trends, customer behavior, and payment methods across different regions and states.

🚀 Project Overview

This dashboard helps in understanding:

Overall Sales & Profit performance
Region-wise analysis
Customer purchasing patterns
Payment method distribution
Product-level profitability

It is designed with a dark theme + gold highlights to give a professional and premium look.
